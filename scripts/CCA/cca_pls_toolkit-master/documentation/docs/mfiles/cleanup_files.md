<span style="font-size:2em;">__cleanup_files__</span>

Cleans up unnecessary duplicate and intermediate files created during
analysis to save disc space.

##  Syntax
      cleanup_files(cfg)
    
##  Inputs
*   **cfg** [*struct*]
    
##  Examples
      % Example 1
      load cfg;
      cleanup_files(cfg);
